# SoloDesk

Privacy-first freelance business dashboard. No cloud, no accounts.

## Features

- Time Tracking
- Client Management
- Project Management
- Income & Expenses
- Invoice Generation

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router DOM

## Getting Started

```bash
npm install
npm run dev
```
